# Cottage client asset provenance

The Cottage client uses a curated subset of the source assets supplied under `raw/`.

* `tiles/house_tiles.png` and `tiles/furniture.png` originate from CozyHome 32x32 Interior v1.0.
* `tiles/interior_free.png` and `characters/*` are used under the same compiled-game-only terms, as confirmed by the project owner.

These files are runtime game assets, not a standalone asset distribution.
